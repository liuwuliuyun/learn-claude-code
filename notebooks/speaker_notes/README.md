# Speaker scripts

One file per notebook, written as **verbatim scripts** for live delivery — sentences you can read aloud, with stage directions in italic and brackets.

The notebooks themselves carry only structural anchors (titles, brief section headers, code cells). The full prose explanation lives here. Open the script and the matching notebook side by side when presenting.

**Total runtime:** ~2.5 hours including Q&A. For a tighter slot, drop Part 4 (error recovery) of lesson 3 and Part 4 (worktrees) of lesson 4 — neither is load-bearing for later lessons.

## Format conventions

- **`> "Say: ..."`** blocks are verbatim — read them aloud.
- **`*[Stage direction]*`** lines tell you what to do (run a cell, switch to the whiteboard, pause).
- Each section ends with a transition cue so you can hand off cleanly into the next part.

## Self-study readers

The notebooks alone are still navigable — the trimmed markdown cells point readers here for the full explanation. If you're following along without an instructor, read the script alongside each notebook.
